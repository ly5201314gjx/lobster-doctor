# Changelog

## v0.1.0 - 2026-03-18

Initial public release of **Lobster Doctor**.

### Added
- Config diagnostics for common OpenClaw ambiguity.
- Skill installability audit with smoke checks.
- Disk risk classification and cleanup guidance.
- Task health diagnostics for long-running AI jobs.
- Aggregate `all` command.
- JSON output and status-based exit codes.
- Baseline / only-new reporting.
- Summary-only and markdown reporting modes.
- Unified summary schema and sorted reporting.
- Automated smoke/analyzer test coverage.
